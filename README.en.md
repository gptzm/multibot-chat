# multibot-chat

#### Description
MultiBot Chat 是一个基于 Streamlit 的多机器人聊天应用，可以同时和多个不同配置的机器人聊天。已支持多种大语言模型（LLM）API，包括 AzureOpenAI、ChatGLM、CoZe、Qwen、Ollama、DeepSeek、Moonshot 和 Yi。
